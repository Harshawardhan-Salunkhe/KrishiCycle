Transforming Waste into Black Gold

Compost Companion is a specialized web application designed to help gardeners and eco-conscious individuals transform kitchen and yard waste into rich soil. Developed by The Cracked HAMS from VIT Pune, this tool provides personalized composting recipes, a marketplace for organic amendments, and a platform for users to sell their homemade compost.

<h2>üöÄ MAIN FEATURES</h2>
<h3>1. Custom Compost Recipe Builder</h3>

    Plant-Specific Guides: Select the type of crops you are growing (Leafy, Root, Fruiting, Flowers, etc.).

    Smart Ingredient Analysis: Checks your "Greens" and "Browns" to calculate the ideal Carbon-to-Nitrogen ratio.

    Step-by-Step Instructions: Detailed preparation and layering guides based on your specific ingredients.

<h3>2. Compost Marketplace</h3>

    Buy: Browse standard blends, vermicompost (worm castings), and organic manures.

    Sell: Local gardeners can create listings, upload photos, and set prices to sell surplus compost.

    Detailed Views: Each product includes a breakdown of benefits and usage descriptions.
